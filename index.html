CSS Property Display Cheatsheet

display

The display CSS property sets whether an element is treated as a block or inline element and the layout used for its children, such as flow layout, grid or flex. Formally, the display property sets an element's inner and outer display types. The outer type sets an element's participation in flow layout; the inner type sets the layout of children content.

Single Value Syntax

Two Value Syntax

Description
display: block; display: block flow; Gives an element block flow. Element starts on a new line and takes up the whole width.
display: flow-root; display: block flow-root; Gives an element block flow. Establishes a new block formatting context containing the elements content.
display: inline; display: inline flow; Gives an element inline flow. Any height and width properties will have no effect.
display: inline-block; display: inline flow-root; Gives the element inline flow. The element can have height and width properties.
display: flex; display: block flex; Gives the element block flow. The element’s content will be displayed using the flex box model.
display: inline-flex; display: inline flex; Gives the element inline flow. The element’s content will be displayed using the flex box model.
display: grid; display: block grid; Gives the element block flow. The element’s content will be displayed using the grid model.
display: inline-grid; display: inline grid; Gives the element inline flow. The element’s content will be displayed using the grid model.

Box Generation

Value

Description
display: none; Turns off the display of an element so that it has no effect on layout (the document is rendered as though the element did not exist).
display: contents; Essentially makes the container disappear, making the child elements children of the element the next level up in the DOM.
